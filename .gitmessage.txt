# 🧵 Commit Message Template
#
# 커밋 메시지 컨벤션을 아래 형식에 맞춰 작성
#
# 형식:
# <gitmoji> <type>: <제목>
#
# 예시:
# ✨ feat: 로그인 기능 추가
# 🐛 fix: 로그인 오류 수정

# Commit Types:
# ✨ feat: 새로운 기능 추가
# 🐛 fix: 버그 수정
# 💡 chore: 기능 변경 없이 사소한 코드 수정 (주석, 오타 등)
# 📝 docs: 문서 수정 (ex. README)
# 🚚 build: 빌드 관련 파일 수정 (설정, 의존성 등)
# ✅ test: 테스트 코드 추가 또는 수정 (프로덕션 코드 변경 없음)
# ♻️ refactor: 기능 변화 없는 코드 리팩터링
# 🚑 hotfix: 운영 중 긴급 수정
# 📦 deps: 의존성 추가 또는 제거
# 🔥 remove: 불필요한 코드 또는 파일 제거
# 🚧 wip: 작업 진행 중 (Work In Progress)
